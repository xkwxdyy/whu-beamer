# whu-beamer

武汉大学 `beamer` 模版.

## 开发背景

历代传下来的旧模版有点古老，不管是中文处理还是宏包堆砌等问题，都不再适用，所以打算使用现代 LaTeX3 开发一个接口清晰的模版，分离格式和内容，让用户能够更专注于内容上。
